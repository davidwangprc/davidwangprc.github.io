## 菜品做法

### 1. 松鼠桂鱼头（镇桌大菜）
{% include figure.liquid path="assets/img/recipes/songshu-head.png" title="松鼠桂鱼头成品" class="img-fluid rounded z-depth-1" %}

1. 鱼头洗净，两面剞菊花刀  
2. 拍干水分 → 撒盐+料酒腌10分钟 → 薄薄裹一层干淀粉  
3. 六成油温下锅炸至定型，捞出再复炸一次至金黄  
4. 番茄酱版/糖醋版汁淋上去即可

### 2. 豆腐鱼片汤（鲜到眉毛掉下来）
{% include figure.liquid path="assets/img/recipes/doufu-soup.png" title="豆腐鱼片汤" class="img-fluid rounded z-depth-1" %}

1. 鱼身片成薄片，用盐、料酒、蛋清、淀粉上浆  
2. 豆腐切片焯水备用  
3. 姜丝爆锅 → 高汤/清水煮开 → 下鱼片滑散 → 放入豆腐 → 胡椒粉、香菜、香油

### 3. 鱼茸蒸蛋羹（嫩得能吸溜当布丁）
{% include figure.liquid path="assets/img/recipes/yurong-egg.png" title="鱼茸蒸蛋羹" class="img-fluid rounded z-depth-1" %}

1. 剩下的鱼肉剁成极细的鱼茸  
2. 加1个蛋清 + 少许盐 + 温水/高汤，高速搅拌上劲  
3. 过筛到碗里，打散2～3个整蛋液拌匀  
4. 水开后中火蒸8～10分钟，淋香油+蒸鱼豉油

### 4. 酸菜鱼尾
{% include figure.liquid path="assets/img/recipes/suancai-fish.png" title="酸菜鱼尾" class="img-fluid rounded z-depth-1" %}

1. 酸菜切丝，干辣椒+花椒备好  
2. 鱼尾拍粉，下锅两面煎至微黄  
3. 另起锅爆香酸菜 → 加汤煮开 → 放入煎好的鱼尾煮5分钟  
4. 撒蒜苗、淋热油激香

### 5. 红烧鱼骨（最下饭）
{% include figure.liquid path="assets/img/recipes/hongshao-bone.png" title="红烧鱼骨" class="img-fluid rounded z-depth-1" %}

1. 鱼骨鱼排斩段，煎至两面金黄  
2. 姜蒜爆锅 → 加入生抽2勺、老抽半勺、糖1勺、料酒、水没过鱼骨  
3. 大火烧开转小火焖15分钟，收汁撒葱花
